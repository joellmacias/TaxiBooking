To use the system, go to booking.html to book a taxi. As an admin go to admin.html, to look at bookings. Enter nothing to get bookings within 2 hours.
Enter a valid booking number to get valid booking details.
Booking.js and booking.php handle the AJAX and processing.
admin.js and admin.php handle the search query and the response.
assign.php handles the assigning of the requests to taxis.
